# Remedio Skills — Authored

Each `skills/<slug>/SKILL.md` is a standalone skill authored in the
[Remedio Skills Hub](https://rony-remedio.github.io/remedio-skills/).

Install any of them in Claude Code via the hub's marketplace:

    /plugin marketplace add https://sowybesirgcejonncnqv.functions.supabase.co/marketplace?c=remedio-all-public
    /plugin install <slug>@remedio

Don't hand-edit files here — use the hub's authoring UI. Direct edits
won't round-trip back into the catalog's metadata.
